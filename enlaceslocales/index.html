Enlaces locales

Los enlaces locales los definimos como los enlaces que realizamos entre nuestras diferentes páginas web de nuestro sitio, incluyendo otros recursos.

Este archivo index.html esta dentro de una carpeta llamada enlacesinternos y dentro de ella se encuentran otros arhivos html, blog y nosotros

Lo que vamos a hacer es enlazar estas tres páginas

Voy a hacer un menú sencilla con una lista desordenada y voy a crear los enlaces internos

  • Index
  • Blog
  • Nosotros

Si ves, podes navegar entre las diferentes páginas de este sitio web usando los enlaces internos, no acedemos a ningún recurso fuera de nuestro sitio, todas las páginas pertenecen al mismo sitio web.
